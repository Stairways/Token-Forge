Deployment Steps
1
Download Your Project ZIP
Your OP-20 contract has been generated as a complete AssemblyScript project. Download the ZIP — it includes your token source code, build config, and all dependencies listed in package.json.
⬇ Download token_project.zip
2
Build the WASM Contract
Unzip the project. In a terminal, run:
npm install
npm run build
This compiles your AssemblyScript token to build/MyToken.wasm. Requires Node.js 18+.
Get Node.js ↗
3
Deploy via OP_WALLET
Open the OP_WALLET browser extension. Switch to OP_NET Testnet. Click Deploy, upload your .wasm file, then sign and broadcast. You need a small amount of BTC for the deployment gas fee.
Install OP_WALLET ↗ Get tBTC ↗
4
Save Your Contract Address
After the transaction confirms, OP_WALLET shows your contract address (starts with op1...). Paste it below to track it on your dashboard.
